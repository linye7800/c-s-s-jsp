### Session 与 Cookie
session 处理用户会话，与登录有关
cookie 也是处理用户会话，与登录有关

session 和cookie的区别
(1) session生成的数据在服务器
    cookie生成的数据在客户端浏览器
(2) 有效期计算方式不同
    session: 有效期从最后一次请求结束后，开始计时
    cookie: 有效期是固定的，从创建开始计时
